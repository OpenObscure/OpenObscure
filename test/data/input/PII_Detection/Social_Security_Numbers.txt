--- Edge cases ---

Multiple SSNs: primary taxpayer 287-65-4321, spouse 452-18-7733, dependent 612-90-3344.

The SSN 078-05-1120 was famously used in a Woolworth wallet advertisement in 1938.

Please do NOT use 000-12-3456 as that is an invalid SSN (area code 000).

Similarly, 666-45-1234 is invalid (area code 666 is reserved).

And 989-12-3456 would be invalid (area codes 900-999 are not issued).

- Employee #1: 154-32-8877
